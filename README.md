# NASA's Algorithms & Datasets

A collection of useful algorithms and datasets for agricultural monitoring, soil moisture analysis, and global weather forecasting.

## Integrated Multi-satellite Retrievals for GPM (IMERG)

[IMERG - Integrated Multi-satellite Retrievals for GPM](https://gpm.nasa.gov/data/imerg)

IMERG provides global estimates of precipitation from a combination of satellite data. It supports rainfall estimation for weather prediction, climate research, and water resource management.

---

## Global Agriculture Monitoring System (GLAM)

[GLAM - Global Agriculture Monitoring System](https://glam1.gsfc.nasa.gov/)

GLAM provides global agricultural monitoring using satellite-based data, helping in the analysis of crop conditions and agricultural production across the world.

---

## Soil Moisture Active Passive (SMAP)

[SMAP - Soil Moisture Active Passive](https://smap.jpl.nasa.gov/)

SMAP is a NASA satellite mission that measures soil moisture and provides data for weather forecasting, flood prediction, and monitoring drought conditions.

---

## FAO's Global Soil Partnership

[FAO's Global Soil Partnership](https://www.fao.org/global-soil-partnership/en/)

The Global Soil Partnership promotes sustainable soil management and the protection of soil resources globally. It offers various datasets and tools for soil monitoring.

---

## Global Forecast System (GFS)

[GFS - Global Forecast System](https://www.ncei.noaa.gov/products/weather-climate-models/global-forecast)

GFS is a global numerical weather prediction system that produces forecasts up to 16 days ahead. It‚Äôs widely used for meteorological analysis and research.

---

## ECMWF's ERA5

[ERA5 - ECMWF's Reanalysis](https://www.ecmwf.int/en/forecasts/dataset/ecmwf-reanalysis-v5)

ERA5 is the fifth generation of ECMWF atmospheric reanalyses of the global climate. It combines model data with observations into a globally complete dataset, helping to analyze climate patterns and forecast future conditions.
